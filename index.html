Upload File to IPFS

IPFS, or InterPlanetary File System, is a protocol and peer-to-peer network for storing and sharing data in a distributed file system. IPFS uses content-addressing to uniquely identify each file in a global namespace connecting all computing devices.

With this we application you can store files in IPFS all you have to do is create and get API key from Pinata.

You can upload images and documents although any web application files like .html might due to piniata restriction plan..
